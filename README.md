Hi 👋 My name is Adolfo Coronado
================================

### Python Developer and Data analyst.


I’m a Mechatronics Engineer passionate about solving complex problems by combining software, electronics, and intelligent control. Over the past few years, I’ve worked as a software developer, tackling challenges in industrial automation, artificial intelligence, and web interface development. My focus is on creating efficient and functional solutions, both in software and hardware. My main interest lies in adaptive control for dynamic systems. I recently developed a self-balancing platform using neural network-optimized PID controllers, a project that resulted in both my thesis and a scientific article focused on the practical application of deep learning in nonlinear systems. This experience strengthened my passion for the convergence between machine learning and embedded systems. Beyond physical system development, I’ve worked as a freelancer on web projects, e-commerce platforms, data analysis, and predictive maintenance using Python and libraries such as Pandas, TensorFlow, and Scikit-learn.
I enjoy learning, collaborating, and building. I’m constantly looking for new projects that challenge me and help me grow as both an engineer and a developer.
